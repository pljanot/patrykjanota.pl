# Readme

Website using hugo templates - that describes 'my' photography projects.

Minimalistc layout, 3 columns | menu | content | images 

Change /content for adding your 'data'
Img, are stored in static/images/


* build into /public dir
```bash
hugo
```

* build and start server using different baseUrl=>localhost:1313
```bash
hugo server -D --baseURL="http://localhost:1313/"
```
